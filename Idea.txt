# Contrastive Learning
 - 같은 행동 영상에 다른 데이터 증강을 적용한 두 샘플은 특징 공간에서 가깝게 만들고, 다른 행동 영상 샘플과는 멀어지도록 모델을 학습시킨다. 이렇게 사전 학습된 모델을 가져와 기존 분류 문제에 미세 조정을 한다.
 - ntu_data_loader에 구현된 다양한 증강 기법을 활용하여 Contrastive Learning 프레임 워크를 구현하고 GCN-Transformer 모델을 사전 학습 시킨 후 분류 테스크에 맞게 파인튜닝 한다.

# 자기회귀 예측과 마스킹 Masked Autoencoding (MAE) 스타일
 - 데이터의 일부를 마스킹하고 모델이 이를 복원하도록 학습시킨다.
 - 입력 시퀀스에서 무작위로 몇 개의 관절을 모든 프레임에서 제거한다. 모델은 주변 관절들의 움직임을 보고 가려진 관절의 7차원 특징 벡터를 예측하도록 학습한다.
 - 무작위로 특정 구간을 통째로 마스킹한다. 모델은 이전과 이후의 움직임을 보고 중간에 무슨일이 있었는지 예측한다.

## 실행 방안
 - 전체 데이터셋 (Training + Validation)을 모두 사용하여 사전 학습을 진행한다.
 - 공간적 마스킹과 시간적 마스킹을 수행한다.
 - 모델은 마스킹된 데이터를 입력 받아, 가리지 않은 주변 시공간 정보를 바탕으로 원본 데이터를 복원하도록 학습한다. 이 때 손실 함수는 원본과 복원된 데이터 간의 MSE를 사용하도록 한다.
 - 이제 사전학습 된 모델을 가져와 행동 분류 문제를 풀도록 소량의 추가 학습을 진행한다.
 - 모델의 가중치를 그대로 불러온 뒤, HEAD를 교체한다. 즉, 데이터 복원을 위해 사용했던 마지막 레이어를 제거하고 그 자리에 60개 행동 클래스를 분류하기 위한 새로운 레이어를 부착한다.
 - 이제 Discriminative Learning Rates를 적용한다. 사전 학습을 통해 모델의 초기 레이어들은 이미 신체의 일반적인 움직임 같은 범용적인 특징을 잘 학습했다. 이 지식은 최대한 보존하는 것이 좋다. 따라서 초기 레이어인 GCN과 초반부 Transformer 블록은 매우 낮은 학습률을 적용하고 후반부 레이어와 새로 부착된 Classification Head에는 상대적으로 높은 학습률을 적용하여 미세조정을 한다.
 - 이제 라벨이 있는 훈련 데이터만을 사용해서 Train.py의 지도 학습 방식을 그대로 따른다. 단, 사전 학습 덕분에 모델이 이미 좋은 출발선에 있으므로, 훨씬 적은 에폭만으로도 더 빠른 성능을 달성할 수 있다.
